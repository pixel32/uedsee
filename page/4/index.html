UX设计原则：系列位置效应

发表于 2021-05-19 评论：

UX设计原则：系列位置效应

摘要：用户最容易记住序列中的第一项和最后一项。

关键词：UX设计，系列位置效应

概述

用户最容易记住序列中的第一项和最后一项。

系列位置效应（英语：serial-position effect）是一种心理学现象，指人们倾向于对首先见到的事物和最后见到的事物有更好的印象。分为首因效应（primacy effect）和近因效应（recency effect）。

起源

系列位置效应是Herman Ebbinghaus创造的一个术语，描述了序列中某个项目的位置如何影响记忆准确性。指人们倾向于对首先见到的事物和最后见到的事物有更好的印象。分为首因效应（primacy effect）和近因效应（recency effect）。

首因效应是对“第一印象”或“先入为主”所产生的心理学现象，指在行为过程中，最先接触的事物会给人留下深刻的感知或认知，影响人对事物的感知和判断。

近因效应指在行为过程中，对事物的最近一次接触给人留下深刻的感知或认知，能对首因效应所形成的心理作用起到巩固、维持、否定、修改或调整等作用。例如在时间上与下一次购买决策最为接近，因此正或负的近因效应能促进或阻滞购买行为。

应用

  • 将最不重要的项目放在列表的中间可能会有所帮助，因为这些项目倾向于较少地频繁存储在长期记忆和工作记忆中。
  • 在诸如导航元素的最左侧和最右侧放置关键按钮可以增强记忆。

UX设计原则：波斯特尔定律

发表于 2021-05-15 评论：

UX设计原则：波斯特尔定律

摘要：你接受什么是自由的，而你发送什么是受限制的（保守的）

关键词：UX设计，波斯特尔定律

概述

波斯特尔法则：你接受什么是自由的，而你发送什么是受限制的（保守的）。

起源

互联网的先驱乔恩·波斯特尔（Jon Postel）制定了波斯特尔定律（也称为稳健性原则）。

该法律是互联网的设计指南，特别是关于TCP和网络的法律，并指出“TCP实现应遵循鲁棒性的一般原则：对自己的工作要保守，对他人的接受要自由”。换句话说，向其他计算机（或同一计算机上的其他程序）发送消息的程序应完全符合规范，但接收消息的程序只要含义清楚就应接受不一致的输入。

应用

  • 对用户可以采取的各种行动或他们可能提供的任何输入抱持灵活态度并容忍他们。
  • 提供可靠且可访问的界面同时，用户在输入，访问和功能方面进行何种操作都有可能。
  • 我们对设计的期望和计划越多，设计的弹性就越大。
  • 接受来自用户的可变输入，翻译该输入以满足系统的要求，定义输入边界，并向用户提供清晰的反馈。

UX设计原则：峰终定律

发表于 2021-05-13 评论：

UX设计原则：峰终定律

摘要：人们判断体验的主要依据是他们在体验的高峰和结束时的感受，而不是体验的每一刻的总和或平均值。

关键词：UX设计，峰终定律，心理学

概述

什么是峰终定律?2002年诺贝尔经济学奖获奖者，心理学家丹尼尔·卡纳曼（Daniel Kahneman）经过深入研究，发现我们对体验的记忆由两个因素决定：高峰（无论是正向的还是负向的）时与结束时的感觉，这就是峰终定律（Peak- End Rule）。

起源

Kahneman，Fredrickson，Charles Schreiber和Donald Redelmeier于1993年进行的一项题为“何时更愿意减轻痛苦：增加更好的结局”的研究提供了突破性的证据。

参与者经历了一次不愉快经历的两个不同版本。第一次试验是让受试者将手浸入14°C水中60秒。第二项试验是让受试者将另一只手浸入14°C水中60秒，然后再将其手浸没30秒，在此期间温度升至15°C。然后向受试者提供重复哪项试验的选择。与时间的单调性定律相反，尽管长时间暴露在不舒适的温度下，受试者还是更愿意重复第二次试验。

应用

  • 密切注意用户旅程中最激烈的时刻和最后时刻（“结束”）。
  • 确定您的产品最有帮助，最有价值或最有趣的时刻，并设计出使最终用户满意的产品。
  • 请记住，人们回忆正面经历比正面经历更生动。

瑞典宜家家居卖场出口处的1元冰淇淋就是一个很好的峰终定律应用。

另外宜家家居几乎所有的家具都需要客户自己组装，组装完成的家具就如同客户自己的作品一样，宜家在客户心目中的好感度会大幅度增加。

UX设计原则：帕金森定律

发表于 2021-05-09 评论：

UX设计原则：帕金森定律

摘要：任何任务都会膨胀，直到所有可用时间都用完为止。

关键词：UX设计，帕金森定律，心理学

概述

帕金森定律：任何任务都会膨胀，直到所有可用时间都用完为止。

起源

西里尔·帕金森（Cyril Parkinson）是英国海军历史学家，他在英国公务员制度中度过了很多时间。作为第二次世界大战中的英国参谋，他观察到由于庞大的官僚机构而造成的许多效率低下。实际上，他指出，即使大英帝国衰落，大英殖民办事处也逐年增加。

在1955年为《经济学人》撰写的讽刺文章中，他讨论了组织由于自身服务的性质而如何不受控制地增长。每个部门都会给另一个部门增加工作。毕竟，每一个官员都想让下属增加而不是让竞争对手增加。

他幽默论文的重点如下：

“工作会自动膨胀，以填补完成工作所需的时间。”

后来他继续用他的书来推广这个观点，我们今天将其称为帕金森定律。半个多世纪后，他的观察仍然是正确的。

应用

帕金森定律不仅适用于官僚机构。它适用于几乎所有工作，尽管在时间管理中更为明显。

  • 在截止期到来之前进行工作会使工作效率更高；在那几个小时之内，您完成的工作比平时要多得多。

  • 不管有多少空间，你的物品都会找到一种占用空间的方式-你下意识地获得了更多物品，以填满未使用的空间。

  • 你把盘子上的所有食物都吃完了。食物的多少以某种方式给你提供大致相同的饱腹感。

解决方案是为生活设定限制和约束。

“我们应该小心，不要将自己的时间用在仅是美好的事情上，而要花很少的时间去做那些更好或最好的事情。” —达林·奥克斯

截止日期是时间的约束。你可以在生活其他方面施加约束，以提高效率。有了这样的约束，你就必须从选配项中筛选出基本要素。

  • 花费太多？必须节省50%的开支，你被迫优先考虑真正使自己开心的事情。

  • 买太多衣服？确保所有衣服都适合你的衣柜。如果有一个新衣服进来，则必须舍弃另一个。

  • 吃得太多？使用较小的容器。当你最终吃了10包迷你薯条而不是家庭装薯条时，您会更加了解自己的进步。

约束迫使我们挑战现状。它使我们重新审视传统的做事方法，并进行改进，以克服出现的挑战。

最终结果是，与没有约束的情况相比，我们可以做得更好，变得更富裕。
